# VSCode

## 全体の設定
* 設定（⌘ + ,） -> ユーザー -> 右上の設定ファイルアイコンをクリック
	* [files/vscode-user-settings.json](https://github.com/norfinnisle/settings/blob/main/vscode/files/vscode-user-settings.json)

## エクスプローラーに表示しないファイルを指定する
* 設定（⌘ + ,） -> ワークスペース -> 右上の設定ファイルアイコンをクリック
	* [files/vscode-workspace-settings.json](https://github.com/norfinnisle/settings/blob/main/vscode/files/vscode-workspace-settings.json)

## カーソルの移動を vim 風にする
* 基本設定 -> キーボードショートカット -> 右上の設定ファイルアイコンをクリック	
	* [files/vscode-keybindings.json](https://github.com/norfinnisle/settings/blob/main/vscode/files/vscode-keybindings.json)
